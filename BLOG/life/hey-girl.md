# 一些和女生交往的技巧

## 把妹达人 30天改造计划

成功条件：三十天内得到约会。约会的定义是跟刚认识的女人约定第二次见面。

### Day 01

+ 自我评估
    * 写下你认为别人怎么看待你的一两个句子
    * 写下你希望别人怎么看待你的一两个句子
    * 列出你自己希望改变的三种行为或个性
    * 列出你自己希望建立的三种行为或个性
+ 闲聊任务
    * 跟五个陌生人闲聊
    * 目标：开启对话
    * 意图：用问题或搞笑填补沉默
    * 答案不重要，跟陌生人讲话就算完成任务
+ 摆脱自我束缚
    * 认识女人这件事情，最大的敌人是自己
    * 越不敢接近女人的人，越不可能被粗鲁地拒绝
    * 别人可能注意你，但未必在评判你，他们也在寻求别人的认同
    * 别把畏缩、心智软弱与好人混为一谈
    * 保持仪容整洁，传达个性即可
    * 真正的自我埋藏在不安全感跟禁忌之下，努力发掘自己最好的一面
    * 没有女人会讨厌亲切风趣诚恳有趣可爱令人舒适又不会一直烦她的人

### Day 02

+ 设定目标
    * 你想达成哪三项成就让自己更快乐
    * 这些成就为什么会让你更快乐
    * 你的个人使命是什么
    * 列举三个完成使命的特定结果
    * 现在你为何全心全意追求个人使命
+ 看着对方的眼睛
    * 找五个陌生人闲聊，记下他们的眼珠颜色

### Day 03

+ 回归原始人生活
    * 别洗澡
    * 别刮胡子
+ 说话有自信
    * 呼吸与姿势
    * 讲话前深呼吸让肺里充满空气
    * 冷静缓慢的声音代表权威
    * 消灭赘字(无用的语气词)，自我纠正
    * 音调与语气的变化
    * 句子结尾不要抬高语调，用斩钉截铁的语气来结束句子
+ 推荐好电影
    * 随便拨打一个电话，对方接听后，试着请对方推荐一部好电影
    * 这个技巧会帮你掌控现实生活中的对话，让它导向你想要的结果
    * 获得三个人的推荐，就算完成任务

### Day 04

+ 好好洗个澡
+ 问专家
    * 请女人介绍附近有卖男装的服饰店
    * 搭讪过三个女人并且获得一个建议就算成功
    * 找看似当地人而且时尚感不错的对象攀谈
    * 跟人在街上说话，不要从后面接近
+ 抬头挺胸
    * 姿势不仅是自信和外表的关键，也与健康有关

### Day 05

+ 该打扮了
    * 问可靠的女性朋友：如果你要选一项来改变我的外表，会是什么
+ 改变装扮
    * 征求免费的造型建议
    * 到店里去找客人最少的时候跟看起来最伶俐的女店员讲话
    * 告诉她你想改变风格，请她帮你搭一套衣服
    * 决定买衣服，顺便问店员可以在哪里买到好鞋子
+ 温习
    * 塑造心智的录音
    * 口语练习
    * 抬头挺胸
+ 丢掉旧衣服
+ 仪容检查表
    * 换发型
    * 别戴眼镜
    * 晒黑
    * 修指甲
    * 去除多余毛发
    * 仔细照镜子
    * 处理眉毛
    * 寻求护肤建议
    * 牙齿美白
    * 改善口臭
    * 带配件
    * 加入健身房
    * 注意饮食
    * 确保衣服合身

### Day 06

+ 克服接近焦虑
    * 害怕的是遭拒
    * 不下场就赢不了
    * 唤醒你的自由心态
    * 改变你的次型态
    * 别在意结果
    * 把失败从字典中剔除
    * 被拒绝策略练习
+ 如果你不会甜言蜜语
    * 随机赞美四个女人，两个是你认识的，两个是陌生人
    * 避免空泛的赞美，赞美特定细节
+ 八小时法则

### Day 07

+ 学习打开组合
    * 不具有威胁性，不让人尴尬
    * 引起好奇心
    * 通往继续对话的跳板
    * 当作展现个性的工具
    * 间接开场白：随机、有趣、跟女人或环境无关的对话
+ 说出开场白之前
    * 不要死盯、品头论足或偷瞄面前的女人
    * 大家都想跟现场最受欢迎的人在一起
    * 发现目标就立刻开始搭讪
    * 刚接近不要正面迎向对方，转头发问，制造出你正要前往别的地方，只是途中暂停随意问个问题，如果没有被拒绝，才转向面对她
    * 不要晃来晃去
    * 接近时要微笑
    * 别怕接近有男人的团体
    * 一定要关照团体中的男人
+ 准备开场白
    * 即兴、出于好奇、对大多数人有趣
    * 想想可能引起大多数人兴趣的话题
    * 只要是真的，再离谱的问题都可能是有效的开场白
+ 测试开场白
    * 接近三个不同的女人，讲出开场白
    * 决不强求
+ 成果评估

### Day 08

+ 微调开场白
+ 用新招接近
    * 自带时间限制
+ 成果评估

### Day 09

+ 检讨时间
+ 接近混合团体
+ 学习的十四个法则
    * 小量获取并运用知识
    * 没有所谓拒绝，只有回馈
    * 女方永远没有错
    * 主动而非被动学习
    * 不要预演负面结果
    * 了解心智的学习模式
    * 愿意经历痛苦期
    * 不寻求亲友的赞同
    * 即使不合理也愿意尝试新观念
    * 一旦某种东西有效，想想为什么会有效
    * 不知道怎么做也不要逃掉
    * 跟比你厉害的人混
    * 确保你的努力成果比例越来越高
    * 有始有终

### Day 10

+ 反向操作日
    * 认识女人后，告诉她你不想把她
+ 假装难搞
    * 欲擒故纵
    * 把她赶走
    * 给自己定价
    * 把她放在朋友区
    * 夸张
    * 角色逆转
    * 雇佣她
    * 装高傲
    * 扮演权威
    * 让她竞争

### Day 11

+ 修饰你的身份
    * 简洁、独特
+ 接近于延续
    * 接近三人以上、至少含有一个女性的团体
    * 使用有时间限制和理由的开场白
    * 讲完之后做下面的事情：
    * 假装要离开，但是只迈出一步
    * 假装好奇发问：你们是怎么认识的？
    * 准备好回应或意见
    * 然后可以走，表示感谢
+ 熟悉你的内心游戏
    * 世界就是你想的样子
    * 没有限制这回事
    * 能量流往注意力所在之处
    * 爱就是快乐
    * 所有力量都来自内心
    * 效果是真理的指标

### Day 12

+ 分享你的特质
    * 写下你希望别人看待你的八个特质
+ 找出自己的故事
    * 讽刺又丢脸的
    * 冒险又刺激的
    * 性感又突兀的
    * 纯真又感人的
    * 琐碎又诗意的
    * 危险又应用的
    * 即时又迷惑的
+ 选择你的故事
+ 准备你的故事
    * 强力的开场
    * 好的结尾
    * 扣人心弦
    * 清晰的细节
    * 加入幽默
    * 加入价值
    * 避免寻求认同
    * 检查故事长度(30s-2min)
+ 陈述故事
+ 表演你的故事
+ 分享你的故事
